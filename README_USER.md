# Hướng Dẫn Sử Dụng - AI Tra Cứu Tài Liệu (Offline)

Chào mừng bạn! Đây là phần mềm giúp bạn tra cứu thông tin từ chính tài liệu của mình bằng trí tuệ nhân tạo (AI), hoạt động hoàn toàn trên máy tính của bạn (không cần internet sau khi cài đặt).

## 1. Cài Đặt Ban Đầu (Chỉ làm 1 lần)

1.  Mở thư mục chứa phần mềm này.
2.  Tìm file có tên **`setup.bat`** (có thể hiển thị là `setup`).
3.  Nhấp đúp chuột (Double-click) vào file đó.
4.  Một cửa sổ màu đen sẽ hiện lên. Hãy làm theo hướng dẫn trên màn hình:
    *   Hệ thống sẽ tự động cài dặt Python và Ollama nếu chưa có.
    *   Hệ thống sẽ tải các mô hình trí tuệ nhân tạo cần thiết (Việc này tốn thời gian tùy tốc độ mạng, hãy kiên nhẫn).
5.  Khi thấy thông báo **"CAI DAT HOAN TAT!"**, bạn có thể tắt cửa sổ đó.

## 2. Cách Sử Dụng Hàng Ngày

1.  Tìm file có tên **`run_app.bat`** (có thể hiển thị là `run_app`).
2.  Nhấp đúp chuột vào file đó.
3.  Chờ một chút, trình duyệt web sẽ tự động mở ra giao diện sử dụng.

## 3. Hướng Dẫn Trong Phần Mềm

### A. Tải Tài Liệu Lên
*   Nhìn sang cột bên trái (Menu).
*   Bấm vào nút **"Browse files"** để chọn file tài liệu của bạn (Hỗ trợ PDF, Word, TXT).
*   Chọn các file xong, bấm nút **"Lưu & Xử lý ngay"**.
*   Chờ hệ thống thông báo "Hoàn tất!".

### B. Hỏi Đáp
*   Nhập câu hỏi của bạn vào ô chat ở dưới cùng màn hình chính.
*   **Ví dụ:** "Tóm tắt nội dung tài liệu?", "Quy trình xin nghỉ phép như thế nào?"
*   AI sẽ trả lời dựa trên tài liệu bạn vừa đưa vào.

## Lưu Ý Quan Trọng
*   Phần mềm chạy **Offline**, dữ liệu của bạn nằm an toàn trên máy tính này.
*   Nếu câu trả lời không có trong tài liệu, AI sẽ trả lời "Tôi không tìm thấy thông tin...".
*   Độ chính xác phụ thuộc vào chất lượng tài liệu bạn cung cấp.
